# Idioma de trabajo — Regla centralizada

## Campos en español
Todos los campos de texto que se insertan en Supabase (concepto, slogan_headline, subtítulo, cta, copy, descripción_corta, buyer_persona, dolor_anhelo, cambio_emocional, diferenciador, beneficios, objeciones_típicas, gatillador) se escriben en **español con ortografía impecable**. Tildes, signos de interrogación/exclamación de apertura y cierre (¿? ¡!), puntuación correcta. Revisar antes de insertar.

## Campo en inglés
**Excepción**: el campo `prompt` se escribe en **inglés** (es lo que recibe el modelo de imagen).
